# Как загрузить новые изменения в текущую ветку на GitHub

## Если проект уже на GitHub и ты хочешь обновить ту же ветку (например main)

Выполни в терминале в папке **pets** (корень проекта):

### 1. Убедиться, что ты на нужной ветке

```bash
git branch
```

Звёздочка покажет текущую ветку (часто это `main` или `master`). Если нужно переключиться на main:

```bash
git checkout main
```

### 2. Добавить все изменения и закоммитить

```bash
git add .
git commit -m "Породы собак, картинки, SQLite, страница детали породы"
```

Текст в кавычках замени на своё описание изменений.

### 3. Отправить изменения на GitHub в эту же ветку

```bash
git push origin main
```

Если твоя ветка называется `master`, подставь её:

```bash
git push origin master
```

---

После этого на GitHub в репозитории обновится та же ветка — новые коммиты появятся там.

## Если remote ещё не добавлен

Если при `git push` пишет, что нет remote:

```bash
git remote add origin https://github.com/ТВОЙ_ЛОГИН/pets.git
```

Потом снова:

```bash
git push origin main
```
